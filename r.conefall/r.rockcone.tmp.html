NAME

r.rockcone - Estimate the area potentially affected by gravitative falls

KEYWORDS

raster, rockfall

SYNOPSIS

r.rockcone
r.rockcone help
r.rockcone elev=string aspect=string start=string deg=float [dir=integer] [fv=string] [mass=string] [freq=string] [vel=string] [ecin=string] [method=max|ave|min] [--overwrite] [--verbose] [--quiet]

Flags:

--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

elev=string
Name of elevation raster map
aspect=string
Name of aspect raster map
start=string
Name of input starting cells raster map
deg=float
Degree of cone inclination from the horizont [0<x<90]
Default: 35
dir=integer
Degree of direction angle range [0<x<360]
Default: 90
fv=string
Velocity factor reduction [0<x<1]
Default: 0.9
mass=string
Mean block mass [Kg]
freq=string
Output frequency raster map
vel=string
Output velocity raster map
ecin=string
Output kinetic energy raster map
method=max|ave|min
Method for cones velocity and energy aggregation
Options: max,ave,min
Default: max
