DESCRIPTION

r.massmov.calibration is developed for automating and simplifying r.massmov model calibration procedure.

The module requires all the r.massmov mandatory inputs. Rheological numerical parameters must be expressed as a range and variation step (min:max:step) or as a list of comma separated values (val,val,val,...). r.massmov optional inputs for automatic stopping criterion (step_thres and stop_thres) and parallel computing (threads) are also available.

Based on input data, the module runs a series of simulations combining all the different values of considered parameters.The possible outputs are:
  • final deposit maps (h+simulation code), optionally based on the same user-defined color table (color_rules);
  • a report (outfile) with basic information (for each simulation: simulation code, used parameters, time length, number of time steps) and other optional statistics (-msc), calculated with respect to the observation map;
  • one or more text files (for each simulation) with coordinates and deposit values along user-defined profiles.

NOTES

Currently the module is available only for Voellmy rheologies.
