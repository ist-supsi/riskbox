NAME

v.sample.class - Sample a raster map by areas,lines or points.

KEYWORDS

vector, raster, sample

SYNOPSIS

v.sample.class
v.sample.class help
v.sample.class input = string raster = string pre = string mode = string [, string ,...] [-- verbose ] [-- quiet ]

Flags:

--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

input=string
Name of input vector map
raster=string
Name of raster basis map
pre=string
Prefix of the new column(s) containing the required statistic(s)
mode=string
Statistic(s) required
Options: min,max,ave,sum
Default: ave

DESCRIPTION

v.rast.stats - samples the statistical parameter of the cells intersected by each single feature of the vector map. Given a vector map, a raster map, a prefix and one or more modes (minimum, maximum, average, sum) this command creates new columns named pre_mode containing the required statistical values.
