NAME

r.dfwalk - Calculate debris-flow hazard map

KEYWORDS

raster, debris-flow, hazard

SYNOPSIS

r.dfwalk
r.dfwalk help
r.dfwalk [-qm] dsm=name start=name [pdir=name] [slim=float] [esp=float] [nrw=integer] [smean=float] [pmd=float] [ps=float] [vl=float] [sl=float] [shm=float] [vhm=float] [sh0m=float] nrrw=name vel=name hgt=name [--overwrite] [--verbose] [--quiet]

Flags:

-q
Quiet
-m
Calculate max velocity instead of mean
--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

dsm=name
Input dem raster
start=name
Input zone file
pdir=name
Input raster preferential directions
slim=float
Slope limit value [%]
Default: 0.36
esp=float
expansion coefficient [-]
Default: 1.8
nrw=integer
number of random walk for each starting cell [-]
Default: 100
smean=float
mean slope of conoid [%]
Default: 0.1
pmd=float
perla mass-to-drag rough coefficient [m]
Default: 70
ps=float
perla slippering rough coefficient [-]
Default: 0.1
vl=float
maximun speed at wich there is deposition [m/s]
Default: 14
sl=float
maximun slope at wich there is deposition [decimal value]
Default: 0.4
shm=float
max deposition possible due to slope in a single cell for a RW [m]
Default: 1.0
vhm=float
max deposition possible due to speed in a single cell for a RW [m]
Default: 1.0
sh0m=float
max volume deposition possible per random walk [m3]
Default: 20
nrrw=name
map random walks
vel=name
raster max velocities
hgt=name
raster sedimentation
